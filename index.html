TOEIC文法ツール
【TOEIC Grammar Tools】

このページでは、TOEICでつまづきやすい文法ポイントを、シンプルなアプリでわかりやすく整理しています。
スコアアップを目指す方に向けて、例文付きで効率よく学べるツールを順次公開中です。

TOEIC -関係詞を学ぶ：関係代名詞・関係副詞・複合関係詞-NEW!

英語の関係代名詞・関係副詞・複合関係詞を感覚的に学べる学習メモアプリです。
文法の違いを例文と翻訳でやさしく理解できるよう設計されています。

TOEIC -前置詞をイメージで学ぶ-

前置詞の使い方を、イラストと例文を通して直感的に理解できる学習ページです。
ボタンを押すことで例文が表示され、空間・時間・方向・移動などの前置詞の違いを穏やかな画面で学べます。

2025年9月15日公開
Demo on GitHub

TOEIC -類義語を学ぶ：前置詞、接続詞、接続副詞-

前置詞・接続詞・接続副詞などを、類義語によって分類しました。
ボタンを押して例文を表示させることにより、使い方の違いを学ぶことができるTOEIC学習ページ です。

TOEIC -つなぎ言葉の違いを学ぶ：前置詞、接続詞、接続副詞-

前置詞・接続詞・接続副詞の違いを穏やかなUIで学べる学習メモアプリです。
代表語と例文をボタン操作で表示でき、noteや技術記録との連携にも適しています。
文法の使い分けを感覚的に理解できる静かな学習空間を提供します。

TOEIC - 語尾で品詞を学ぶ

このツールは、品詞の語尾を分類・表示し、例文とともに学べる学習空間です。
語尾の種類を選ぶと、対応する例文が表示されます。
やさしい配色とシンプルな構成で、集中しやすく、感覚的に学べるよう設計されています。
