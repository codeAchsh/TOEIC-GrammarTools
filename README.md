# TOEIC文法ツール（TOEIC Grammar Tools）

このプロジェクトは、TOEIC学習者向けに文法や語尾に関するつまづきやすいポイントを、シンプルなWebアプリとしてまとめたものです。  
例文付きで、品詞の語尾や文法構造を直感的に学べます。

今後は、同様の学習アプリを追加しながら、まとめて販売できる形に整えていく予定です。  
現在は無料公開中ですが、学習支援を目的とした個人利用に限ります。
